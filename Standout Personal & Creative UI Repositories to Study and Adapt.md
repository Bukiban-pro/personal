# Standout Personal & Creative UI Repositories to Study and Adapt
## Overview
This report focuses on personal portfolios, creative landing pages, and animation-heavy experiences where individual developers and small teams push aesthetics, motion, and interaction beyond generic templates.
All entries are open-source repositories or clearly linked source for live sites, so they can be cloned, dissected, and adapted as learning material.
## How This Differs From Generic UI Kits
- **Real-world, opinionated aesthetics:** These projects reflect a specific creator’s taste and narrative rather than a neutral design system, which makes them better for learning how to build a "vibe" rather than only components.[^1][^2]
- **Heavy use of motion and 3D:** Many examples lean into Framer Motion, GSAP ScrollTrigger, React Three Fiber, and custom scroll effects to create cinematic experiences.[^3][^4][^1]
- **Highly specific structures:** Instead of reusable kits, these repos show full experiences: hero storytelling, about sections, project galleries, 3D scenes, micro-interactions, and scroll flows wired together.[^5][^2]
## High-Impact React / Next.js Portfolios with Motion
These projects are personal or small-team portfolios with strong animation, transitions, and composition.
### Framer Motion–Driven Portfolios
- **codebucks27/react-portfolio-final** – A "stunning portfolio" in React using styled-components and Framer Motion for page transitions, animated sections, and hover effects; built as a full site, not just a kit.[^6]
- **gianlucajahn/portfolio** – A TypeScript + React portfolio that explicitly showcases UI/UX expertise, using SASS and Framer Motion for refined section reveals, transitions, and content choreography.[^7]
- **afrinxnahar/my-portfolio-react** – A responsive personal portfolio using React, Tailwind CSS, and Framer Motion, offering a clean layout and subtle motion that can be re-themed heavily.[^8]
- **mo-hassann/developer-portfolio** – A Next.js portfolio styled like a code editor that uses Framer Motion for smooth transitions and interactive skill/project sections, combining a playful concept with modern motion.[^9]
- **Next.js + Tailwind + Framer Motion portfolio templates (e.g., JS Mastery)** – Curated lists of best Tailwind portfolio templates call out a Next.js + Tailwind + Three.js + Framer Motion portfolio as one of the most visually impressive free templates, featuring bento grid layouts and an interactive 3D globe.[^10]
- **codebucks27/Next.js-Developer-Portfolio-Starter-Code** – A Next.js + Tailwind + Framer Motion portfolio that is visually rich and meant to be customized, combining a modern landing experience with reusable sections.[^11]
### Three.js / React Three Fiber Portfolios
- **adrianhajdin/threejs-portfolio** – A 3D minimalistic portfolio built with React.js, Three.js, and Tailwind CSS, designed specifically to "create a lasting impact" with 3D scenes embedded into a clean layout.[^1]
- **adrianhajdin/project_3D_developer_portfolio** – A 3D developer portfolio project focused on teaching how to bring 3D graphics and animation into a portfolio site, mirroring what many high-end marketing sites do.[^3]
- **teshank2137/portfolio** – A React + ThreeJS portfolio emphasizing interactive 3D elements that respond to user input, embedded in a responsive portfolio structure.[^5]
- **jrefusta/joan-portfolio** – A personal portfolio from a 3D developer showcasing GLSL shaders, Three.js, and game-like interactions, featuring playful elements like interactive Rubik’s cubes and arcade machines and distributed as a full open-source site.[^12][^2]
- **chiragbadhe/ThreeJs-Portfolio** – A Next.js + Three.js + Tailwind portfolio featuring a controllable 3D soldier model and other interactive elements, blending character interaction with a traditional portfolio structure.[^13]
- **3D portfolio topics & showcases** – three.js forum posts and GitHub topics like `3d-portfolio` and `creative-portfolio-websites` surface multiple personal experiments in 3D-heavy sites with public source code.[^14][^15][^16]
### GSAP / Scroll-Heavy Portfolios
- **vipulkumar-dev/portfolio** – A Next.js portfolio explicitly built around GSAP animations, focusing on scroll and entrance effects beyond typical Tailwind transitions.[^17]
- **afzalimdad9/animated-portfolio** – A creative animated portfolio website using Next.js, with the author explicitly encouraging people to fork and reuse it, making it ideal for inspecting animation decisions.[^18]
- **lumeudio-next (from Reddit)** – An open-source animated Next.js template for agencies and creative portfolios using shadcn/ui, Tailwind, Motion.dev, and Motion Primitives for smooth page transitions and modern motion patterns, shared publicly with an MIT license.[^19]
## Scroll, Parallax, and Storytelling Landing Pages
These projects focus on scroll-based storytelling, parallax, and cinematic sequences.

- **kostastepetes/gsap-scroll-trigger-3d-landing-page** – A GSAP ScrollTrigger–driven landing page with parallax slides and scroll-linked 3D-style animations, showing how to orchestrate a multi-section story purely via scroll position.[^20]
- **waleed46shah/project_smoothCats** – A Next.js landing page using GSAP, ScrollTrigger, and Locomotive Scroll to achieve the kind of smooth scrolling and micro-interactions common on Awwwards-style sites, explicitly inspired by such experiences.[^4]
- **mayank1305j/Gsap-Scrolltrigger-Landing-Page** – A "beautiful landing/portfolio style page" tuned with GSAP ScrollTrigger and Textillate.js, featuring interactive sections that react to horizontal swipes and hover interactions.[^21]
- **akhtarhssn/animate-on-scroll (Scroll Animation Website)** – A multi-section scroll animation site using JavaScript and CSS to demonstrate a variety of section-based reveals and interactions in a simple but extensible form.[^22]
- **scroll-animation GitHub topic** – The `scroll-animation` topic collects dozens of small but creative examples of scroll-driven animations, sliders, and dashboards with code that can be lifted and recomposed.[^23][^24]
## 3D and R3F Landing Experiences
Beyond portfolios, some repos focus on 3D marketing-style landing pages.

- **ashgole/r3f_landing_page** – A React Three Fiber + Next.js landing page showcasing 3D pet models with interactive exploration, combining a modern layout with real-time 3D scenes.[^25]
- **Aanyaa26/Three.js-landing-page** – A 3D car showcase built with React Three Fiber and Three.js that presents car models in an interactive environment, closer to product marketing visuals than a simple portfolio.[^26]
- **React-Three-Fiber-Boilerplate (Sean-Bradley)** – A multi-branch boilerplate covering many R3F techniques, with branches that mimic sophisticated scenes and interactions found in commercial experiences.[^27]
- **pmndrs/react-three-fiber** – The core React renderer for Three.js, whose examples and ecosystem (via the `react-three-fiber` topic) are full of advanced scene setups, camera rigs, and controls used in visually striking sites.[^28][^29]
## Meta-Discovery: Finding More "Da Vinci"–Level Projects
Instead of only relying on named repos, certain GitHub Topics and curated collections act as portals into hundreds of creative projects.

- **Topics to explore:** `nextjs-portfolio`, `nextjs-portfolio-template`, `portfolio-websites`, `personal-portfolio`, `open-source-portfolio`, `creative-portfolio-websites`, `3d-portfolio`, `scroll-animation`, `gsap-scrolltrigger`, and `react-three-fiber` all contain real-world, often ambitious projects.[^30][^14][^23][^28][^31][^32][^33][^34][^16]
- **Curated lists and roundups:** Articles and roundups like "20 Best Tailwind CSS Portfolio Templates" actively cherry-pick the most visually impressive free templates from GitHub and elsewhere, highlighting which repos stand out in practice.[^10]
- **Community showcases:** three.js forum "Showcase" posts often link directly to GitHub repos for personal 3D portfolios and experiments, providing both a polished live site and the exact source that powers it.[^12][^2][^15]
## How to Study These Like an Apprentice
- **Trace the animation stack:** When you open a repo, identify whether it uses Framer Motion, GSAP + ScrollTrigger, Lenis/Locomotive, or React Three Fiber, then follow how entry points, hooks, and config are organized across pages and components.[^1][^17][^4]
- **Reverse-engineer layout and rhythm:** Look at spacing, typography, and content order in standout portfolios (e.g., 3D scenes paired with simple text and whitespace) and re-create them with your own content to internalize the pacing.[^5][^2][^1]
- **Steal patterns, not skins:** Fork a project like `joan-portfolio` or a GSAP landing page, strip all branding, then gradually re-theme colors, type, imagery, and motion curves while keeping the underlying structure and interaction logic.[^20][^4][^2]
- **Build a private "pattern library" repo:** As you dissect these projects, copy your favorite sections (hero, about, project grid, 3D scene wrapper, scroll section) into your own internal library so you can recombine them into new, original experiences.
